# Hello, Instrktr!

Welcome to your first step. This is a sample step to verify everything works.

## Task

Create a file called `hello.txt` in the workspace root with the following content:

```text
Hello, Instrktr!
```

You can create the file from the terminal:

```bash
echo "Hello, Instrktr!" > hello.txt
```

Or create it manually — open [hello.txt](open:hello.txt) and type the text.

When you're done, click **Check** in the panel below.
