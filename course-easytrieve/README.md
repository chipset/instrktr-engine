# Easytrieve with Code4z: Generate, Validate, and Execute

An [Instrktr](https://marketplace.visualstudio.com/items?itemName=instrktr.instrktr) course based on the workflow for **Easytrieve Language Support**, **JCL Language Support**, and **Zowe Explorer** in VS Code.

You practice the same artifacts and terminology you use on a real system: Easytrieve source, execution JCL, upload settings, and how to read job output in Zowe Explorer.

## What you'll learn

| Step | Topic |
|------|--------|
| 1 | Prerequisites: Zowe profile and the three Code4z extensions |
| 2 | A minimal Hello World Easytrieve program |
| 3 | How the editor supports syntax, hover/autocomplete, and navigation |
| 4 | JCL that runs Easytrieve (`EZTPA00`) with standard DDs |
| 5 | `eztdsn` / `jcldsn` settings and the upload → submit flow |
| 6 | Finding the job, opening DDs, and interpreting completion codes |

**Total: ~1 hour** (plus time on your LPAR if you run jobs for real)

## Prerequisites

- VS Code 1.93 or later
- Instrktr extension installed
- Optional but recommended: Zowe CLI or Zowe Explorer configured with a working mainframe profile to execute the full workflow outside the course exercises

## Getting started

1. Open the **Instrktr** panel in the VS Code activity bar
2. Find **Easytrieve with Code4z** in the catalog
3. Click **Install**, then **Start Course**
