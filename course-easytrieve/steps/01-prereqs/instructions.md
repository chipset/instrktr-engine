# Prerequisites and Code4z Extensions

Modern Easytrieve development in VS Code relies on **Code4z** extensions that connect editing, JCL validation, and the mainframe.

## What you need first

Before you generate JCL or upload members, you should have:

1. A **Zowe profile** (or equivalent connection) so VS Code can reach your mainframe.
2. **Easytrieve Language Support** — editing, diagnostics, JCL generation, upload helpers.
3. **JCL Language Support** — validate job control before submission.
4. **Zowe Explorer** — browse data sets, submit jobs, and read spool output.
