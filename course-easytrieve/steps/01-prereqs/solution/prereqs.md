# Easytrieve workflow — prerequisites

## Zowe profile

My Zowe profile stores host, user, and connection details so VS Code extensions can list data sets, upload members, and submit jobs against my LPAR.

## Extensions installed

- Easytrieve Language Support
- JCL Language Support
- Zowe Explorer

## Why validate JCL first

JCL Language Support flags syntax errors, missing or misaligned DD statements, and invalid parameters so I fix them locally instead of failing on the mainframe.
