# Syntax, Symbols, and Navigation

Easytrieve Language Support gives you the same class of help you expect from a modern language server:

| Capability | What it does for you |
|------------|----------------------|
| **Syntax and diagnostics** | Highlights structure and surfaces errors while you type. |
| **Hover and autocomplete** | Shows field definitions and suggests valid keywords. |
| **Go to Definition** | Jumps between symbols and macro sources pulled from the host when configured. |

## The exercise

Edit [hello.ezt](open:hello.ezt) and add **three comment lines** (each line starting with `*` in column 1 is a good habit on mainframe sources). Each comment must reflect one row in the table above:

1. One line mentioning **syntax** or **diagnostic** (or both).
2. One line mentioning **hover** or **autocomplete** (or both).
3. One line mentioning **definition** or **macro** (or both).

These comments are only for this course exercise — they document what you will use in the real extension.

Click **Check My Work** when done.
