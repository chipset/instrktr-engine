# After the job runs

## Finding output

Use the **Jobs** section in the Zowe Explorer side bar, filter by user or job id, then expand the job node.

## Reading the listing

Expand the job’s DD list and open **SYSPRINT** (or another DD) — VS Code opens the spool text in a new tab.

## Success signal

A condition code of **0000** on the job indicates normal completion for the example workflow.
